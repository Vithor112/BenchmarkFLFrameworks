Seu algoritmo proposto é um que usa Denoising Diffusion Probabilistic Models para aumentar o tamanho dos datasets locais. Esse DDPM funciona adicionando ruído gaussiano as imagens num processo de markov e depois ele reverte esse ruído, usando uma rede neural, para aproximar a imagem pré-ruido. Treinar com dados gerados artificialmente reduz a perfomance do modelo. Para lidar com isso eles adicionam o conceito de soft label (label smoothing technique), transformando os labels em vetores probabilisticos para cada categoria aumentando a entropia dos labels gerados.
